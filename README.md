 Expense Tracker in Python

A CLI-based Expense Tracker application built using Python and Object-Oriented Programming (OOP).  
This project allows users to add, view, delete, and calculate total expenses with permanent data storage using JSON.

---

 Features

- Add Expense
- View Expenses
- Delete Expense using Serial Number
- Calculate Total Expense
- JSON File Storage
- Menu-Driven Interface
- Object-Oriented Programming (OOP)

---

 Technologies Used

- Python
- JSON
- OOP Concepts
- File Handling
